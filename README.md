## Описание
Конвертер цветов из HEX в RGB. При вводе корректного HEX-кода меняет цвет фона и показывает RGB-представление. При ошибке выводит предупреждение.

## Стек
- React
- TypeScript
- Vite

## Установка и запуск
```bash
yarn install
yarn dev
